## Decisions Made

- `robots.txt` uses `sitemap-index.xml` not `sitemap.xml` — the @astrojs/sitemap integration generates `sitemap-index.xml` as the index entry point; `sitemap.xml` would result in a 404 for Google
- OG placeholder is generated from pure Node.js `Buffer` + `zlib` with no external npm packages — keeps the generator self-contained
- No changes to `astro.config.mjs` — `site: 'https://redbirdlawnservice.com'` and `sitemap()` were already in place from Phase 01 foundation work
