STEP 1 — Create src/components/astro/SectionWrapper.astro (new file, directory may not exist — create it):

```astro
---
// src/components/astro/SectionWrapper.astro
// Wraps any content block in a data-animate div for scroll-triggered fade-up.
// The IntersectionObserver in Layout.astro watches all [data-animate] elements
// and adds .in-view when they reach 15% threshold.
// Usage: <SectionWrapper delay={200}><YourSection /></SectionWrapper>

interface Props {
  class?: string;
  delay?: number;
  tag?: string;
}

const { class: className = '', delay, tag: Tag = 'div' } = Astro.props;
---
<Tag
  data-animate
  data-animate-delay={delay}
  class={className}
>
  <slot />
</Tag>
```

NOTE: Astro does not support dynamic tags via variable (Tag = 'div') in all versions. If `<Tag>` syntax causes a build error, fall back to `<div data-animate ...>`. Test build after creation.

STEP 2 — Update src/layouts/Layout.astro to add IntersectionObserver script before closing </body>:

Add this script block immediately before `</body>`:

```astro
    <!-- Scroll animation observer — fires once per element at 15% threshold -->
    <!-- Per VISL-02: fade-up 20px, 0.6s, IntersectionObserver -->
    <script>
      const observer = new IntersectionObserver(
        (entries) => {
          entries.forEach((entry) => {
            if (entry.isIntersecting) {
              entry.target.classList.add('in-view');
              observer.unobserve(entry.target);
            }
          });
        },
        { threshold: 0.15 }
      );

      document.querySelectorAll('[data-animate]').forEach((el) => {
        observer.observe(el);
      });
    </script>
```
